# Dialog

Built with the [Radix UI Dialog](https://www.radix-ui.com/primitives/docs/components/dialog) primitive.

<Canvas of={Stories.Primary} />

## Example

```tsx
import { Dialog } from '@nearai/ui';

...

<Dialog.Root>
  <Dialog.Trigger asChild>
    <Button label="Open Dialog" />
  </Dialog.Trigger>

  <Dialog.Content title="My Dialog">
    <Text>Some content here.</Text>
    <Text>More content over there.</Text>
  </Dialog.Content>
</Dialog.Root>
```
